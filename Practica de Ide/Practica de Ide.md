# Práctica: Instalación de diversos IDEs

Para la correcta realización de la siguiente práctica, realizar una captura de pantalla de cada uno de los siguientes pasos principales.

---

## 1. Instalación de Eclipse
![Alt text](image.png)
1. Ir a la página oficial: [Eclipse IDE for Java Developers](https://www.eclipse.org/downloads/).
2. Seleccionar la versión acorde al sistema operativo.
3. Ejecutar el instalador.
4. Seleccionar **Eclipse IDE for Java Developers**.
5. Elegir carpeta de instalación.
6. Elegir y confirmar un **workspace** (carpeta de trabajo).

---

## 2. Comprobación del funcionamiento de Eclipse

1. Crear un nuevo proyecto de Java:  
   **File → New → Java Project**
2. Nombre: `ProyectoPrueba`
3. Escribir el siguiente código:

```java
public static void main(String[] args) {
    System.out.println("Eclipse funciona correctamente");
}
